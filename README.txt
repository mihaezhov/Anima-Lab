Anima Lab — внутренний инструмент для подбора и фиксации фирменного motion-стиля.
Вместо зоопарка bezier-кривых мы используем spring-модель, где характер движения (ζ = dampingRatio) выбирается дизайнерами, а длительность автоматически считается от размера элемента.

Основные идеи

Один мастер-spring для всей экосистемы.

duration определяется по размеру (sizePx) по таблице или формуле.

Из ζ и duration система считает stiffness и damping.

Токен версионируется (spring@v1, spring@v2) и хранится в дизайн-системе.

Разработка вызывает springFromTokens(sizePx) и получает готовые параметры/сниппеты для Web, iOS и Android.

Зачем

Нет десятков кривых «под каждый случай».

Нет ручного «обрезания» easing-кривых.

Консистентность во всех приложениях.

Простота: один токен = один стиль.

Расширения

При необходимости можно ввести дополнительные токены (spring@soft, spring@snappy), но на старте достаточно одного master-spring.
